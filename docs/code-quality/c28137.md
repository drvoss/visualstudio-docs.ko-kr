# <a name="c28137"></a>C28137
경고 C28137: 변수 인수가 대신 (리터럴) 상수 여야 합니다.

 함수 호출이 필요한 상수(리터럴)가 없는 경우 이 경고가 보고됩니다. 함수에 대한 설명서를 참조하세요.

## <a name="example"></a>예제
 예를 들어 [ExAcquireResourceExclusiveLite](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdm/nf-wdm-exacquireresourceexclusivelite) 루틴에는 `Wait` 매개 변수의 값이 TRUE 또는 FALSE 여야 합니다. 다음 코드 예제에서는 이 경고를 생성합니다.

```
ExAcquireResourceExclusiveLite(Resource, Wait);
```

 다음 코드 예제에서는 이 경고를 방지합니다.

```
ExAcquireResourceExclusiveLite(Resource, TRUE);
```
