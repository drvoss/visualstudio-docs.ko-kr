# <a name="c28163"></a>C28163
경고 C28163: 함수를 try/except 블록 내에서 호출 해서는 안 됩니다.

 이 경고는 함수가 `try/except` 블록에 포함 되지 않아야 하는 형식인 경우 `try/except` 블록에 있는 경우에 보고 됩니다. 함수가 호출한 하나 이상의 경로에서 발견된 코드 분석 도구가 `try/except` 블록 안에 있습니다.
