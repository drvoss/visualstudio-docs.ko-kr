# <a name="c28196"></a>C28196
경고 C28196: 요구 사항이 충족 되지 않았습니다. 식이 true로 계산 되지 않습니다.

 이 경고는 분석 중인 함수의 함수 프로토타입에 `__notnull` 매개 변수 또는 반환 값에 대해 `__null`, `__drv_valueIs` 또는 `_Out_`가 포함되어 있지만 반환된 값이 해당 주석과 일치하지 않음을 나타냅니다.
