# <a name="c6059"></a>C6059
경고 C6059: @No__t-0function > 호출에서 길이 매개 변수가 잘못 되었습니다. @No__t-0variable의 버퍼 크기가 아닌 나머지 문자 수를 전달 >

 이 경고는 문자열 연결 함수에 대 한 호출이 연결할 문자 수에 대해 잘못 된 값을 전달 하는 것일 수 있음을 나타냅니다. 이 오류로 인해 악용 가능한 버퍼 오버런 또는 충돌이 발생할 수 있습니다. 이 오류의 일반적인 원인은 버퍼의 남은 문자 수가 아니라 버퍼 크기를 문자열 조작 함수에 전달 하는 것입니다.

## <a name="example"></a>예제
 다음 코드에서는이 경고를 생성 합니다.

```
#include <string.h>
#define MAX 25

void f( )
{
  char szTarget[MAX];
  char *szState ="Washington";
  char *szCity="Redmond, ";

  strncpy(szTarget,szCity, MAX);
  szTarget[MAX -1] = '\0';
  strncat(szTarget, szState, MAX); //wrong size
  // code ...
}
```

 이 경고를 해결 하려면 다음 코드에 표시 된 것 처럼 올바른 문자 수를 사용 하 여 연결 합니다.

```
#include <string.h>
#define MAX 25

void f( )
{
  char szTarget[MAX];
  char *szState ="Washington";
  char *szCity="Redmond, ";

  strncpy(szTarget,szCity, MAX);
  szTarget[MAX -1] = '\0';
  strncat(szTarget, szState, MAX - strlen(szTarget)); // correct size
  // code ...
}
```

 Safe 문자열 조작 함수를 사용 하 여이 경고를 해결 하려면 다음 코드를 참조 하세요.

```
#include <string.h>

void f( )
{
  char *szState ="Washington";
  char *szCity="Redmond, ";

  size_t nTargetSize = strlen(szState) + strlen(szCity) + 1;
  char *szTarget= new char[nTargetSize];

  strncpy_s(szTarget, nTargetSize, szCity,strlen(szCity));
  strncat_s(szTarget, nTargetSize, szState,
                    nTargetSize - strlen(szTarget));
  // code ...
  delete [] szTarget;
}
```
