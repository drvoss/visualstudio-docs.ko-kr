# <a name="c6230"></a>C6230

경고 C6230: 의미 체계가 다른 정수 형식 간의 암시적 캐스트입니다. 부울 컨텍스트에서 HRESULT를 사용 합니다.

이 경고는 `if` 문과 같이 부울 결과가 필요한 컨텍스트에서 `HRESULT`이 사용 됨을 나타냅니다. 이 테스트는 잘못 된 결과를 생성할 수 있습니다. 예를 들어 `HRESULT` (`S_OK`)에 대 한 일반적인 성공 값은 부울로 테스트할 경우 false입니다.

## <a name="example"></a>예제

다음 코드에서는이 경고를 생성 합니다.

```cpp
#include <windows.h>

VOID f( )
{
  LPMALLOC pMalloc;
  HRESULT hr = CoGetMalloc(1, &pMalloc);

  if (hr)
  {

    // code ...
  }
  else
  {
    // code ...
  }
}
```

대부분의 경우에는 [SUCCEEDED](/windows/desktop/api/winerror/nf-winerror-succeeded) 또는 [FAILED](/windows/desktop/api/winerror/nf-winerror-failed) 매크로를 사용 하 여 `HRESULT`의 값을 테스트 해야 합니다. 이 경고를 해결 하려면 다음 코드를 사용 합니다.

```cpp
#include <windows.h>

VOID f( )
{
  LPMALLOC pMalloc;
  HRESULT hr = CoGetMalloc(1, &pMalloc);

  if (SUCCEEDED(hr))
  {

    // code ...
  }
  else
  {
    // code ...
  }
}
```

이 경고의 경우 `SCODE` 형식은 `HRESULT`로 처리 됩니다.
