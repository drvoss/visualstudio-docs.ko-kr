# <a name="c6306"></a>C6306
경고 C6306: \<function >에 대 한 호출이 잘못 되었습니다. va_list를 인수로 수락 하는 \<function >를 사용 하는 것이 좋습니다.

 이 경고는 잘못 된 함수 호출을 나타냅니다. @No__t_0 패밀리에는 가변 인수 목록을 사용 하는 여러 함수가 포함 되어 있습니다. 그러나 `va_list` 인수를 사용 하 여 이러한 함수를 호출할 수는 없습니다. 이러한 호출에 사용할 수 있는 해당 `vprintf` 기능 패밀리가 있습니다. 잘못 된 인쇄 함수를 호출 하면 잘못 된 출력이 출력 됩니다.

## <a name="example"></a>예제
 다음 코드에서는이 경고를 생성 합니다.

```
#include <stdio.h>
#include <stdarg.h>

void f(int i, ...)
{
   va_list v;

   va_start(v, i);
   //code...
   printf("%s", v); // warning 6306
   va_end(v);
}
```

 이 경고를 해결 하려면 다음 코드를 사용 합니다.

```
#include <stdio.h>
#include <stdarg.h>

void f(int i, ...)
{
   va_list v;

   va_start(v, i);
   //code...
   vprintf_s("%d",v);
   va_end(v);
}
```
