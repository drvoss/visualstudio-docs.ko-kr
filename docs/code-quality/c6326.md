# <a name="c6326"></a>C6326
경고 C6326: 상수가 다른 상수와 비교 될 수 있습니다.

 이 경고는 중복 코드 인 다른 상수와 상수가 비교 될 수 있음을 나타냅니다. 사용자의 의도가 코드에서 제대로 캡처되고 있는지 확인 해야 합니다. 경우에 따라 테스트 조건을 단순화 하 여 동일한 결과를 얻을 수 있습니다.

## <a name="example"></a>예제
 다음 코드는 두 개의 상수가 비교 되기 때문에이 경고를 생성 합니다.

```
#define LEVEL
const int STD_LEVEL = 5;

const int value =
#ifdef LEVEL
10;
#else
5;
#endif

void f()
{
  if( value > STD_LEVEL)
  {
    // code...
  }
  else
  {
    // code...
  }
}
```

 다음 코드는 #ifdef 문을 사용 하 여이 경고를 해결 하는 한 가지 방법을 보여 줍니다. 실행 해야 하는 코드를 결정 합니다.

```
#define LEVEL
const int STD_LEVEL = 5;

const int value =
#ifdef LEVEL
10;
#else
5;
#endif

void f ()
{
#ifdef LEVEL
  {
    // code...
  }
#else
  {
    // code...
  }
#endif
}
```
