# <a name="c6335"></a>C6335
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

경고 C6335: \<handlename > 프로세스 정보 핸들 누수  
  
 이 경고는 함수의 CreateProcess 패밀리에서 반환 된 프로세스 정보 처리를 CloseHandle을 사용 하 여 닫아야 함을 나타냅니다. 이렇게 하지 않으면 핸들 누수가 발생 합니다.  
  
## <a name="example"></a>예  
 다음 코드에서는이 경고를 생성 합니다.  
  
```  
#include <windows.h>  
#include <stdio.h>  
  
void f( )  
{  
    STARTUPINFO si;  
    PROCESS_INFORMATION pi;  
  
    ZeroMemory( &si, sizeof(si) );  
    si.cb = sizeof(si);  
    ZeroMemory( &pi, sizeof(pi) );  
  
    // Start the child process.   
    if( !CreateProcess( "C:\\WINDOWS\\system32\\calc.exe",  
                        NULL,    
                        NULL,               
                        NULL,                
                        FALSE,               
                        0,                   
                        NULL,                
                        NULL,                
                        &si,    // Pointer to STARTUPINFO structure.  
                        &pi ) ) // Pointer to PROCESS_INFORMATION  
  {  
    puts("Error");  
    return;  
  }  
    // Wait until child process exits.  
    WaitForSingleObject( pi.hProcess, INFINITE );  
    CloseHandle( pi.hProcess );  
}  
```  
  
 이 경고를 해결 하려면 `CloseHandle (pi.``hThread)`를 호출 하 여 다음 코드와 같이 스레드 핸들을 닫습니다.  
  
```  
#include <windows.h>  
#include <stdio.h>  
  
void f( )  
{  
    STARTUPINFO si;  
    PROCESS_INFORMATION pi;  
  
    ZeroMemory( &si, sizeof(si) );  
    si.cb = sizeof(si);  
    ZeroMemory( &pi, sizeof(pi) );  
  
    // Start the child process.   
    if( !CreateProcess( "C:\\WINDOWS\\system32\\calc.exe",  
                        NULL,    
                        NULL,               
                        NULL,                
                        FALSE,               
                        0,                   
                        NULL,                
                        NULL,                
                        &si,    // Pointer to STARTUPINFO structure.  
                        &pi ) ) // Pointer to PROCESS_INFORMATION  
    {  
      puts("Error");  
      return;  
    }  
  
    // Wait until child process exits.  
    WaitForSingleObject( pi.hProcess, INFINITE );  
    CloseHandle( pi.hProcess );  
    CloseHandle( pi.hThread );  
}  
```  
  
 자세한 내용은 [CreateProcess 함수](https://msdn.microsoft.com/library/ms682425.aspx) 및 [CloseHandle 함수](https://msdn.microsoft.com/library/ms724211.aspx)를 참조 하세요.
