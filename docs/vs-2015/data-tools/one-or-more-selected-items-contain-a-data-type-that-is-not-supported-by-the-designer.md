# <a name="one-or-more-selected-items-contain-a-data-type-that-is-not-supported-by-the-designer"></a>선택한 항목 중 하나 이상이 디자이너에서 지원되지 않는 데이터 형식을 포함하고 있습니다.
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

@No__t_3에**데이터베이스 탐색기** / **서버 탐색기** 에서 끌어 온 항목 중 하나 이상이 [!INCLUDE[vs_ordesigner_short](../includes/vs-ordesigner-short-md.md)]에서 지원 하지 않는 데이터 형식 (예: [CLR 사용자 정의 형식](https://msdn.microsoft.com/library/9f70e0b0-3a0d-4eb1-b914-07a5d0c167c2))을 포함 합니다.

### <a name="to-correct-this-error"></a>이 오류를 해결하려면

1. 지원되지 않는 데이터 형식이 들어 있지 않은 테이블을 기초로 뷰를 만듭니다.

2. 뷰를 **서버 탐색기** /**데이터베이스 탐색기** 에서 디자이너로 끌어 옵니다.
