# <a name="string-expected"></a>문자열이 필요합니다.
@No__t_2이 아닌 형식의 개체에 대해 **valueOf** **메서드를 호출** 하려고 한 경우입니다. 이 호출 형식의 개체는 `String` 형식 이어야 합니다.  
  
### <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- @No__t_2 형식의 개체에 대해서만 **valueOf** **메서드를 호출 합니다.**
